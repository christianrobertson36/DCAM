# DCAM v17 Technician and Engineer Profiles

## Goal

Add the first protected technician and engineer management foundation.

## Added

- staff profile database records linked to users
- staff qualification records
- protected staff API
- People menu item and page
- profile search and filters
- profile create/edit form
- skills, service areas, working hours and availability
- qualification entry with issue and expiry dates
- expiring qualification summary count
- audit records for profile and qualification changes

## Permissions

- `staff:view`
- `staff:edit`
- `staff:admin`

The frontend hides unavailable actions, and the API enforces access.

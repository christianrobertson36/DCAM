# DCAM v46 Bilingual Help Centre

## Purpose

The Help Centre explains DCAM in simple, non-technical language for new and occasional users.

## Languages

- English
- Romanian

The Help Centre follows the language selected in DCAM Settings. Article content is authored separately in both languages rather than relying on automatic translation.

## Included guides

- first ten minutes in DCAM
- navigation and language
- customers and Customer 360
- pipeline, quotations and contracts
- assets
- work orders and technician jobs
- service requests and defects
- compliance inspections and forms
- reports and certificates
- users, permissions, branding and language
- basic troubleshooting

## User experience

- top-bar Help Centre access for immediate visibility
- searchable plain-language articles
- topic filters
- expandable step-by-step instructions
- short practical tips
- responsive phone, tablet and desktop layout
- role reminder explaining why some buttons may not be visible

## Access control

The Help Centre requires `dashboard.view`, making it available to every authenticated DCAM role while keeping it unavailable before login.

## Adding future help

Every new major module should add or update a Help Centre article in both English and Romanian. Instructions should:

1. use short sentences
2. describe the exact menu and button names
3. explain the correct order of actions
4. mention permission-related differences
5. avoid unexplained technical terminology

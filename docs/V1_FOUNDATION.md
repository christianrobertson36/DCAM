# DCAM v1 Foundation Notes

## Project Name

DCAM - Digital Compliance & Asset Management

## v1 Goal

Create the working technical foundation for the app:

- API service
- Frontend service
- PostgreSQL service
- Docker Compose
- Health endpoint
- Admin shell
- Module placeholders

## First Build Check

Backend:

npm install
npm run check

Frontend:

npm install
npm run build

Docker:

docker compose build
docker compose up -d

# DCAM

Digital Compliance & Asset Management

DCAM is a Technical Compliance Operating System for managing customers, buildings, assets, compliance, engineers, work orders, reports, certificates, contracts, renewals and dashboards.

## v1 Foundation

Includes:

- Backend API foundation
- Frontend admin shell
- PostgreSQL-ready Docker setup
- Health endpoint
- Module placeholders
- Documentation folder
- Fixed image tag deployment approach
